# Fashion-MNIST Benchmark

Benchmarking a **convolutional neural network** against a **Random Forest**
(scikit-learn, with optional HOG features) on the
[Fashion-MNIST](https://github.com/zalandoresearch/fashion-mnist) dataset.

The goal is a clean, reproducible comparison of a deep model and a classical
model on the same 10-class image classification task — comparing accuracy,
per-class precision/recall, and training cost.

## Status

Early scaffold. Tooling and structure are in place; the implementation
(data, models, evaluation) is being added.

## Setup

```bash
# install uv first: https://docs.astral.sh/uv/getting-started/installation/
uv sync --extra dev      # creates .venv and installs everything
```

## Structure

```
fashion-mnist-benchmark/
├── pyproject.toml        # dependencies (uv)
├── uv.lock               # pinned versions (generated by `uv lock`)
├── README.md
├── .gitignore
├── fmnist/               # the package — all real logic
│   ├── __init__.py
│   ├── data.py           #   FashionMNIST loading + DataLoaders
│   ├── cnn.py            #   the CNN model + training
│   ├── rf.py             #   Random Forest (+ optional HOG)
│   └── evaluate.py       #   confusion matrix, precision/recall, plots
├── notebooks/
│   └── report.ipynb      # imports fmnist/, runs, displays results
├── data/                 # auto-downloaded Fashion-MNIST (gitignored)
└── outputs/              # figures, saved models, metric tables (gitignored)
```
